Restaurant Management System (Python + SQLite)

A command-line based restaurant management system built using Python and SQLite.
This project includes user login, category management, menu item management, 
and role-based access (admin/customer).

 Features
- login
- Admin and customer roles
- Add / View / Delete categories
- Add / View / Update / Delete menu items
- View menu,place order (customer)
- SQLite database storage
- Preloaded default users and menu

 Technologies Used
- Python 3
- SQLite3
  
 How to Run
1. Install Python.
2. Download  this repository.
